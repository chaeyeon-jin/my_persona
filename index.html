페르소나 카드 선택

당신에게 맞는 페르소나 카드를 선택하고 출력하세요.

Solo Entrepreneur

"내가 만든 건 언젠가 서비스가 된다."

Solo Entrepreneur

  • 작은 기능도 창업 서비스의 씨앗
  • 혼자서도 기획·개발·디자인까지 책임지는 태도
  • 완벽보다 실행 우선

Hacker

"작동하는 게 최고다."

Hacker

  • 문제를 만나면 끝까지 해결한다.
  • 빠르게 프로토타입을 만든다
  • 완벽한 코드보다 돌아가는 카드를 우선한다

Designer

"사람이 쓰기 편해야 진짜 완성이다."

Designer

  • UX/UI를 책임진다
  • 사용자 중심으로 설계한다
  • 팀의 디자인 완성도를 끌어올린다

Engineer

"안정된 구조가 좋은 서비스의 힘이다."

Engineer

  • 데이터 흐름과 기능 구현 책임
  • 버그를 잡고, 안정적인 코드 유지
  • 팀의 기술적 신뢰를 만든다

Storyteller

"이 서비스는 왜 필요한가?"

Storyteller

  • 프로젝트의 비전과 가치를 설명한다
  • 발표·브랜딩·피칭을 맡는다
  • 아이디어를 외부에 잘 전달한다

Explorer

"새로운 도구와 길을 두려워하지 않는다."

Explorer

  • 새로운 기술과 트렌드를 먼저 시도한다
  • 팀에 영감을 주고 호기심을 자극한다
  • 익숙하지 않은 걸 시도하는 용기를 갖는다
